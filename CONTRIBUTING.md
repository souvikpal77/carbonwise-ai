# Contributing to CarbonWise AI

Thank you for your interest in contributing to CarbonWise AI.

## How to Contribute

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Test your changes.
5. Submit a pull request.

## Contribution Guidelines

* Write clean and readable code.
* Follow project structure and naming conventions.
* Update documentation when necessary.
* Ensure the application builds successfully before submitting changes.

## Reporting Issues

If you find a bug or have a feature request, please create an issue describing the problem and possible solution.

We welcome contributions that help improve sustainability awareness and environmental impact tracking.
