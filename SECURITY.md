# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in CarbonWise AI, please report it responsibly.

## Supported Versions

The latest version of CarbonWise AI receives security updates and improvements.

## Security Practices

* Dependencies are regularly reviewed.
* Sensitive information is not stored in the repository.
* Security issues will be investigated and resolved as quickly as possible.

Thank you for helping keep CarbonWise AI secure.
